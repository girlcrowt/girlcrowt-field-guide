## Phase Brief

This is the first true construction phase of your Lab.

Until now, every step has prepared the environment.

Now you begin creating the systems that future Adventures will build upon.

Take your time.

Build carefully.

Document every meaningful decision.

The quality of your Lab is determined by the habits you establish now.

---

## The Goal

Create a stable, well-documented virtual machine that serves as the foundation for future exploration.

Remember:

The objective is not simply to finish.

The objective is to build something you can trust.

---

## Suggested Workflow

### Create

Create the virtual machine according to your Mission Plan.

---

### Install

Install the chosen operating system.

---

### Configure

Apply the initial configuration required for your Lab.

---

### Verify

Confirm that the system starts correctly and behaves as expected.

---

### Document

Record the configuration, installation choices and any observations made during the build.

---

## Checkpoint

Before continuing to Phase 05, ask yourself:

- Does the virtual machine operate as expected?
- Can I explain every major configuration choice?
- Is the system documented?
- Would I be comfortable rebuilding this machine if necessary?

If not, improve the system before moving on.

And remember: Documentation is a kindness to your Future Self!

The expedition continues when the first machine is ready.

# Communication

## The Question

> **How does information move?**

Systems cannot function without communication.

Components exchange information.

People exchange ideas.

Processes exchange data.

The Communication Realm explores how information is created, transmitted, received and understood.

---

## Purpose

The Communication Realm helps you understand how information flows within and between systems.

It examines the paths information follows, the methods used to exchange it and the obstacles that may prevent successful communication.

Whether information moves between computers, services or people, the same principles apply.

---

## What Belongs Here?

Examples include:

- network communication
- protocols
- application programming interfaces (APIs)
- client-server communication
- requests and responses
- messaging systems
- logs
- error messages
- documentation
- diagrams
- command-line output
- user interfaces
- conversations
- meetings
- handovers

If information moves from one place to another, it belongs here.

---

## What Does Not Belong Here?

The Communication Realm does not primarily explain:

- which components exist;
- how components are related;
- whether communication should be trusted;
- how knowledge is preserved;
- whether a solution meets its requirements.

Those questions belong to other Realms.

---

## Typical Questions

- Who communicates with whom?
- What information is exchanged?
- Why is this communication necessary?
- How does the information travel?
- What happens if communication fails?
- Which information is missing?
- Which communication is unnecessary?
- Which assumptions depend on successful communication?
- How can communication become clearer?

## Relationships with Other Realms

The Systems Realm identifies the components.

The Relationships Realm explains how they are connected.

The Communication Realm explains how information moves between them.

Examples:

- Trust asks whether communication can be relied upon.
- Expression asks how information is represented.
- Knowledge asks what information should be preserved.
- Evaluation asks whether communication achieves its intended purpose.

---

## The Goal

A system cannot function without communication.

Understanding how information moves is essential to understanding how the system itself works.
